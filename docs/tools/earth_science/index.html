Earth science

  • SimPEG - An open source python package for simulation and gradient based parameter estimation in geophysical applications.
  • pyGIMLi - pyGIMLi is an open-source multi-method library for modelling and inversion in geophysics.
  • ObsPy - A Python Toolbox for seismology/seismological observatories.
  • Fatiando a Terra - An open toolbox for the Geosciences.
    • Verde: Gridding, machine learning style.
    • Pooch: Easily download datasets.
    • Harmonica: All things potential fields.
    • Boule: Ellipsoids and normal gravity.
    • Ensaio: Practice datasets to probe your code.
